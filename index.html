Combinator is something it explains the relationship between selectors.There are Four different combinators in CSS.

  • descendant selector(space)
  • child selector(>)
  • adjascent sibling selector(+)
  • general sibling selector(~)
  1. descendant selector(space):This selector matches all elements that are descendants of a specified element.
  2. child selector(>):child selector selects all the elements that are children of a specified element.
  3. adjascent sibling selector(+):It is used to select an element that is directly after another specific element.Sibling elements must have same parent element and adjascent means immediatly following.
  4. general sibling selector(~):it selects all elements that are next siblins of a specified element.
